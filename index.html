Unit 3

In this unit I started to add CSS to my portoflio. I created a color palette using basic color theory and applied it to my website. I also chose fonts that would be easy to read and applied them to my paragaphs. I also got a refresher on how the box model works. Using IDs and classes, I was also able to arrange specific elements in a way that makes the page easy to read.

Some things I learned

  • How to position elements using the float property
  • How to style a navigation bar
  • How Pseudo-class selectors work
